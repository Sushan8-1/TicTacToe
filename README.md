Tic Tac Toe 🎮

A modern and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.
This project focuses on clean design, smooth gameplay, and simple game logic while keeping the code lightweight and beginner-friendly.

The game runs directly in the browser without requiring any frameworks or external libraries.

Features
Interactive Tic Tac Toe board
Responsive design for desktop and mobile
Dynamic turn switching
Winner detection system
Draw detection
Restart / reset game functionality
Clean and minimal UI
Smooth hover and click effects
Fast and lightweight performance
Built With
HTML for the structure
CSS for styling and layout
JavaScript for game logic and interactivity
