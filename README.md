# Tic-Tac-Toe

This is a simple Tic-Tac-Toe game created using HTML, CSS, and JavaScript.  
The project was made for practice and learning basic web development concepts, especially JavaScript.

While building this game, I learned how to work with functions, event listeners, conditions, arrays, and DOM manipulation. It also helped me understand how game logic works in JavaScript.

The design of the project is kept simple and clean so the game is easy to play and understand. Players can take turns marking boxes until one player wins or the match ends in a draw.

This project helped me improve my problem solving skills and gave me more confidence in creating interactive websites using JavaScript.

In the future, I would like to improve this project by adding:
- A mode to play against the computer
- Score tracking system
- Restart and reset improvements
- Better animations and sounds
- Mobile friendly layout improvements

This project was created as a learning project and practice for frontend web development.
